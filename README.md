# Pi-hole Blocklists

Lists of domains to be blocked by [Pi-hole](https://github.com/pi-hole/pi-hole).

## Deployment

Just copy the raw URL of the domain list and add it to your [Pi-hole](https://github.com/pi-hole/pi-hole).
